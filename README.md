## Profile Card Component
A responsive and accessible Profile Card built using vanilla HTML, CSS, and JavaScript.
This project focuses on semantic structure, accessibility, and clean UI design while meeting strict testing requirements using data-testid attributes

### Features
- Fully responsive profile card layout
- Semantic HTML structure (article, section, nav, figure)
- Live display of current epoch time in milliseconds
- Accessible avatar with descriptive alt text
- Social links opening in new tabs
- Separate sections for hobbies and dislikes
- Keyboard-accessible interactive elements
- Clean, modern UI with subtle hover effects

### Tech Stack
HTML5 (semantic markup)
CSS3 (Flexbox, responsive design)
Vanilla JavaScript (time update logic)
